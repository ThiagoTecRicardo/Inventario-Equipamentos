# 📦 Inventário de Equipamentos

Sistema de inventário para notebooks corporativos, desenvolvido em **Java + Spring Boot**, **Angular** com suporte a **PostgreSQL** e execução via **Docker Compose**.  
O objetivo é centralizar e gerenciar informações de equipamentos de TI de forma simples e escalável.

---
## 🚀 Instalação e Configuração

### Pré-requisitos
- **Java 17+**
- **Maven**
- **Docker** (para banco de dados PostgreSQL)
- **Angular** (para Interface Grafica)
- **Node.JS** 

---
### Clonar o repositório
```bash
git clone https://github.com/ThiagoTecRicardo/Inventario-Equipamentos.git
cd Inventario-Equipamentos
```
### Navegue nas sub pastas 'backend e frontend' para pegar as intruções de execução das aplicações

### JAVA
```bash
cd /backend 
```
### Angular
```bash
cd /frontend-angular 
```
